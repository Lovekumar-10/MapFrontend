A **full-stack interactive map application** that allows users to **share their live location** and **find routes** between two points. Built with **React**, **Leaflet**, and **Node.js/Express**, this project combines real-time map functionality with a clean and responsive UI.

## Overview

This project is a **full-stack map application** designed for practical location sharing and route planning. Users can:

- Share their **live location** in real-time  
- Find **routes** between two locations  
- View the route visually on a **Leaflet map** with markers and polylines  
- Interact with a **clean, responsive UI**  

It is built to demonstrate the combination of **React frontend**, **Leaflet map library**, and **Node.js/Express backend** integration.

---

## Problem Statement

Most map applications are either **static** or rely heavily on third-party paid services. Users need an easy-to-use interface to:

- Share their **current location**  
- Find routes without complicated setup  
- Have a responsive web interface for both desktop and mobile  

Existing tools often lack **real-time interaction**, making them unsuitable for practical usage in small projects or learning environments.

---

## Solution / Approach

This project solves the problem by:

- **Displaying a dynamic map** using Leaflet and React-Leaflet  
- **Allowing live location updates** that users can share  
- **Calculating routes** between any two points with interactive markers and polylines  
- Creating a **responsive and user-friendly UI** with Tailwind CSS  
- Using a **Node.js/Express backend** to manage route and location data (if needed for future real-time updates)  

---

## Features
